## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/Nichols-AI/Nichols-AI-Agents.git
cd Nichols-AI-Agents
poetry install

# Launch the UI
poetry run streamlit run ottomator-agents/crawl4AI-agent/streamlit_ui.py
```
